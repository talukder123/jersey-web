# Talukder Jersey Shop

A modern football jersey e-commerce website built with HTML and CSS. The site features a FIFA World Cup 2026-inspired design, responsive layout, and a clean shopping experience for football fans.

## Features

- 🏆 FIFA World Cup 2026-inspired UI
- 👕 Jersey collection showcase
- 🛒 Shop page with product categories
- 📱 Responsive design
- ⚡ Hosted with GitHub Pages

## Technologies Used

- HTML5
- CSS3
